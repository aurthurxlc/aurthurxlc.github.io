java不变性小结

一旦 String 对象在内存（heap区）中被创建了，它就不能被改变；

如果我们需要创建可变的 String 对象， 我们可以使用 StringBuffer 或者 StringBuilder ，否则很多时间可能被浪费在垃圾回收上，因为每一次都会有一个新的对象产生。
